# Git Comandi importanti

## Git Init

```bash
git init
```

Serve per iniziare un progetto Git.

## Git Status

```bash
git status
```

Serve per vedere lo stato del progetto.

## Git Add

```bash
git add .
```

Serve per aggiungere tutti i file al repository.

```bash
git add <file>
``` 

Serve per aggiungere un singolo file al repository.
